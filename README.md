# Viajes Doñana - Aplicación de Escritorio

Aplicación de escritorio desarrollada con **JavaFX** para el **Ayuntamiento de Almonte**, como parte del proyecto **Viajes Doñana**. La aplicación permite la **gestión y consulta de viajes y viajeros**, conectándose a la **API RESTful Viajes Doñana** para el acceso centralizado a los datos.

Además, la aplicación ofrece funcionalidades avanzadas como la **generación de informes** y la **visualización de estadísticas**, facilitando el análisis y control de la información registrada.

## Características

- Aplicación de escritorio desarrollada con **JavaFX**.  
- Conexión con la **API RESTful Viajes Doñana**.  
- Gestión y consulta de **viajes y viajeros**.  
- Registro, edición y visualización de datos.  
- **Generación de informes** exportables.  
- **Visualización de estadísticas** sobre viajes y viajeros.  
- Interfaz gráfica intuitiva y orientada al uso administrativo.  
- Optimizada para uso interno del **Ayuntamiento de Almonte**.
